# PS4-GoldHEN-Plugin-Installer
A clean and simple Windows tool for installing PS4 plugins over FTP 
via GoldHEN. Built for the No Remorse GTA V community.

## Features
- Connect to your PS4 via FTP
- Upload .prx plugin files directly
- Automatic plugins.ini configuration
- Conflict detection and safe deactivation
- Manual plugin manager
- One-click revert to original config

## Requirements
- PS4 with GoldHEN installed
- FTP enabled on your PS4
- Windows 10/11

## How to use
1. Download the latest release below
2. Run PluginInstaller.exe
3. Enter your PS4 IP address and port (default: 2121)
4. Select your .prx file and click Upload
5. Done!
